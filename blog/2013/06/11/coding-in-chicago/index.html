Coding in Chicago

11 June 2013

Got moved into my apartment in Chicago, and have been coding as much as possible. Micah, from my DBC cohort, lives in the same building so I have been working with him a bit. I also went to Coding Coffee, which was pretty cool, got a bit done on my Imgur / Reddit scraper, and had some shitty Starbucks coffee. Its been really hot here, which I dont mind so much but its also been extremely humid, which I really dont like, luckily the apartment has really good AC.
