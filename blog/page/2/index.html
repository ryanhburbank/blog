Today was my second day at DevBootcamp in Chicago. It was simply awesome.

A Few Thoughts on Pair Programming

24 June 2013

So.. Pair Programming.

Today was my first real shot at it. Im not sure what to think of it, but I am sure that if we are learning how to do it properly, its good. I thought it was pretty good, at some points and it was mostly dependant on who I was pairing with. Some people it really was pretty good, but then a couple it was just eh, and I dont blame anyone but myself.

I am so used to working on my own and in my own little zone, when I see a possibility of improving the code or something I just jump on the keyboard and starting layin down code. Which certainly isnt how pair programming works.

Ill keep posting on pair programming as I get better / or worse at it. I would really like to get really understand, sooner than later, so I dont hurt my partners learning while pairing, but for now we are all new to it.
